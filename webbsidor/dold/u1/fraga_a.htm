Fråga A1: Beskriv allmänt vad ett protokoll är i datakommunikation.

Ett protokoll är en formell standard för kommunikation på ett nätverk. Det består av en samling regler som beskriver hur kommunikationen på nätverket ska fungera. Dessa regler handlar oftast om hur datan ska paketeras, formatteras och hanteras.

Fråga A2: I kommunikation på webben används bl.a. protokollen HTTP, TCP och IP. Beskriv vilken funktion dessa har.

HTTP
 I nätverks sammanhang talar man ofta om OSI modellen. Denna modell består av 7 olika abstraction layers. HTTP protokollet är ett application layer protokoll vilket betyder att det definerar hur kommunikationen på nätverket ska fungera. HTTP protokollets största ansvar är att beskriva hur hypermedia dokument (som t.ex. HTML) ska skickas.

TCP
 TCP är ett transport layer protokoll och används för dataöverföring. Detta är det vanligaste protokollet att använda för datakommunikation på internet. TCP skapar en anslutning mellan två datorer och använder sig av en dataström för att skicka paketerad data mellan datorerna.

IP
 IP är ett network layer protokoll och arbetar tillsammans med TCP. IP definerar hur strukturen på paketen ska se ut och hur paket ska addresseras för att de ska kunna skickas och bli mottagna rätt.
